# Scientific Graph Studio

> Create publication-quality scientific graphs from experimental data in minutes.

Scientific Graph Studio is a modern web application designed to help undergraduate STEM students generate professional scientific graphs quickly and accurately without relying on spreadsheets or complex scientific software.

The project aims to simplify the process of visualizing experimental data by allowing users to import or paste datasets, generate interactive graphs, perform regression analysis, and export high-quality figures suitable for laboratory reports.

---

## Project Goals

The primary goals of Scientific Graph Studio are to:

- Reduce the time students spend creating scientific graphs.
- Provide an intuitive alternative to Excel for laboratory graphing.
- Generate publication-quality graphs suitable for academic reports.
- Help students visualize experimental data accurately.
- Serve as a learning project to strengthen modern full-stack software development skills.

- Calculate and display R²

## Why I Built This

During undergraduate laboratory courses, students often spend more time learning spreadsheet software than analyzing their experimental results. Scientific Graph Studio was created to provide a simpler, faster, and more intuitive way to generate professional scientific graphs for laboratory reports while also serving as a platform to strengthen my full-stack software engineering skills.
